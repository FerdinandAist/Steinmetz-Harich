# Steinmetzwerkstatt Harich Website

Statische, datensparsame Website mit lokalen Bildern, Kontaktformular via Mailprogramm, mobiler Kontaktleiste, SEO-Metadaten, Breadcrumbs, LocalBusiness-Schema, Sitemap und rechtlich zu prüfenden Platzhalterseiten für Impressum und Datenschutz.

Zum lokalen Prüfen einen statischen Server im Projektordner starten und die Startseite öffnen.
